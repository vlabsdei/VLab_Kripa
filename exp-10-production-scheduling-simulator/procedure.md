# Procedure

## Step 1 – Initialize the Production Scheduling Simulator

Launch the Production Scheduling Simulator. The simulator loads the production floor containing multiple machines, an incoming job queue, production scheduler, performance dashboard, and scheduling controls.

---

## Step 2 – Observe the Initial Factory Configuration

Observe the default production parameters displayed in the Factory Control Panel.

The default parameters include:

- Job Arrival Rate
- Number of Machines
- Machine Processing Speed
- Scheduling Algorithm

Also observe the initial values of the Production Performance Indicators before starting the simulation.

---

## Step 3 – Configure Production Parameters

Modify one or more production parameters using the control panel.

Configure the following parameters:

- Job Arrival Rate
- Number of Active Machines
- Machine Processing Speed
- Scheduling Algorithm

Observe how these parameters influence the scheduling process.

---

## Step 4 – Select a Scheduling Algorithm

Choose one of the available scheduling algorithms.

The simulator supports:

- First Come First Served (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling
- Round Robin

Each scheduling algorithm allocates production jobs differently.

---

## Step 5 – Start the Production Simulation

Click the **Start Simulation** button.

The simulator begins generating production jobs, placing them into the production queue according to the selected Job Arrival Rate.

The Production Scheduler continuously assigns waiting jobs to available machines.

---

## Step 6 – Observe Job Scheduling

Observe the movement of production jobs through the manufacturing workflow.

Study the following sequence:

- Job Arrival
- Queue Formation
- Job Selection
- Machine Allocation
- Job Processing
- Job Completion

Observe how different scheduling algorithms influence job execution.

---

## Step 7 – Monitor Machine Utilization

Observe the status of each production machine.

Analyze:

- Busy Machines
- Idle Machines
- Current Processing Job
- Processing Progress

Compare machine utilization under different scheduling algorithms.

---

## Step 8 – Observe Queue Behaviour

Monitor the Production Queue throughout the simulation.

Observe:

- Queue Length
- Waiting Jobs
- Job Assignment
- Queue Reduction

Compare queue behavior under different production loads.

---

## Step 9 – Analyze Production Performance

Observe the Production Performance Dashboard.

Analyze the following indicators:

- Machine Utilization
- Makespan
- Average Waiting Time
- Production Throughput
- Production Efficiency
- Queue Length

Study how scheduling decisions influence factory performance.

---

## Step 10 – Compare Scheduling Algorithms

Repeat the simulation using different scheduling algorithms.

Compare:

- Job Completion Time
- Average Waiting Time
- Machine Utilization
- Queue Length
- Production Efficiency

Determine which scheduling algorithm performs best for different production conditions.

---

## Step 11 – Identify Production Bottlenecks

Increase the Job Arrival Rate or reduce the Number of Active Machines.

Observe:

- Queue Growth
- Machine Overloading
- Increased Waiting Time
- Reduced Throughput

Analyze how bottlenecks affect overall production performance.

---

## Step 12 – Review Production History

Observe the Production History generated during the simulation.

Analyze:

- Job ID
- Assigned Machine
- Arrival Time
- Start Time
- Completion Time
- Waiting Time
- Job Status

Use the recorded information to compare scheduling performance.

---

## Step 13 – Reset the Experiment

Click the **Reset** button.

The simulator clears all production jobs, resets performance indicators, restores default parameters, and prepares the production scheduling system for a new simulation.

Repeat the experiment using different scheduling algorithms and production configurations to compare their effectiveness and identify the optimal scheduling strategy.
