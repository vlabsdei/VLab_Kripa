### About the Experiment

The Production Scheduling Simulator is a virtual laboratory experiment developed under the SmartFactory 4.0 Virtual Lab project. This experiment introduces students to the principles of production scheduling used in modern Industry 4.0 manufacturing systems. Students learn how production jobs are assigned to multiple machines using different scheduling algorithms and how scheduling decisions influence overall manufacturing performance.

The experiment provides an interactive simulation where users can configure production parameters such as job arrival rate, number of active machines, machine processing speed, and scheduling algorithm to analyze their impact on machine utilization, queue length, average waiting time, makespan, production throughput, and production efficiency. Through real-time visualization of the production floor, students gain practical knowledge of intelligent scheduling techniques that improve productivity and optimize manufacturing resources in smart factories.
