### Procedure

1. Open the Industrial Sensor Monitoring Simulation.

2. Observe the default values of Temperature, Pressure, Vibration, and Humidity.

3. Adjust the Temperature slider and observe changes in machine health.

4. Modify the Pressure parameter and analyze its effect on system performance.

5. Change the Vibration level and monitor the generated machine status.

6. Vary the Humidity level and observe the impact on overall machine health.

7. Experiment with different combinations of sensor values.

8. Click the Calculate button to generate machine health results.

9. Observe warning messages and health indicators.

10. Record observations for different operating conditions.

11. Compare normal and abnormal machine conditions.

12. Reset the simulation if required and repeat the experiment.

### Result

The experiment demonstrates how industrial sensors are used to continuously monitor machine conditions and assist in predictive maintenance through machine health analysis.
