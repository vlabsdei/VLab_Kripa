### About the Experiment

The Industrial IoT Network Security Simulator is a virtual laboratory experiment developed under the SmartFactory 4.0 Virtual Lab project. This experiment introduces students to the principles of Industrial Internet of Things (IIoT) communication, industrial networking, and cybersecurity in modern Industry 4.0 manufacturing environments.

The experiment provides an interactive simulation where users can configure industrial network parameters such as device count, packet transmission rate, firewall strength, attack intensity, communication protocol, and encryption level to analyze their impact on packet loss, network latency, throughput, network availability, and overall security score. Students gain practical knowledge of secure industrial communication, Industrial IoT architectures, cybersecurity mechanisms, firewall protection, and real-time network monitoring used in smart manufacturing systems.
