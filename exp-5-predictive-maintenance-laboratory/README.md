### About the Experiment

The Predictive Maintenance Laboratory is a virtual laboratory experiment developed under the SmartFactory 4.0 Virtual Lab project. This experiment helps students understand the principles of predictive maintenance by simulating machine health monitoring using industrial sensor parameters such as motor temperature, vibration frequency, operating hours, and lubrication level.

The experiment provides an interactive simulation where users can modify machine operating parameters and observe their impact on system health, failure probability, and maintenance scores through a real-time digital twin visualization.
